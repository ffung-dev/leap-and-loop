# leap & loop website

welcome to the leap & loop website!

leap & loop is a teen-run  business founded by me and my friend jenny (hi jenny) that creates crochet plushies/things and custom commissions! we're based on long island in ny (but when we go to college we'll likely expand location... see "locations" page on our website)
this website was created to showcase our crochet projects, share our events, highlight our community impact, and make it easier for people to connect with us.

## about the project

this website is more than just a business page!! leap & loop combines creativity, entrepreneurship, and community involvement through handmade crochet, workshops, and service projects.

the goal of this website is to:
- showcase our crochet work
- share upcoming events and locations
- highlight our connection with the gnshs fiber arts club
- make it easy for customers and community members to contact us

## features

- responsive website for desktop and mobile
- project gallery showcasing crochet creations
- dynamic events and locations
- gnshs fiber arts club information page
- contact form
- content managed through sanity cms
- reusable components built with next.js

## tech stack

- next.js
- typescript
- tailwind css
- sanity cms
- vercel

## content management

this website uses sanity cms so that content can be updated without changing code.

sanity manages:
- projects
- events
- locations
- images
- website text
- club information

this allows leap & loop to continue growing and adding new content over time :)
